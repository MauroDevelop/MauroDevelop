### Actualmente trabajando en...

#### 🎓 [Backend Development Path 2026](https://github.com/MauroDevelop/learning-path-2026)
Estoy documentando todo mi proceso de aprendizaje avanzado en este repositorio, aplicando buenas prácticas y patrones de diseño. Mis avances y aplicaciones incluyen:

- **Arquitectura y Core:** Desarrollo con **TypeScript** y **Node.js**. Implementación de Arquitectura en capas, Programación Orientada a Objetos (OOP) e Inyección de Dependencias.
- **APIs y Datos:** Servidores RESTful con **Express v5**, gestión de base de datos con **Prisma ORM** (y transacciones) conectando a **MySQL/TiDB**, y manejo de archivos con **Multer** y **Cloudinary**.
- **Seguridad y Calidad:** Autenticación y autorización mediante **JWT**, **Bcrypt** y políticas **RBAC** (Roles). Validación estricta de datos con **Zod**, y configuración de **CORS**.
- **Testing y DevOps:** Pruebas unitarias y de integración utilizando **Jest** y **Supertest**. Documentación de endpoints con **OpenAPI/Swagger**, y contenedorización con **Docker** para despliegues en **Render**.
